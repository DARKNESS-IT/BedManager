# МебельТорг — Учёт кроватей

Бесплатное приложение для управления каталогом кроватей. Работает на **Windows** и **Android**. Единый CSV-файл для обеих платформ.

---

## Возможности

- **Полный каталог** — 14 полей: модель, производитель, цвет, спальное место, высота, длина, материал каркаса, основание, конструкция, изголовье, количество, цена, видимость.
- **Фильтрация** — по 10 параметрам: производитель, цвет, спальное место, высота, длина, материал, основание, конструкция, изголовье, диапазон цен.
- **Сортировка** — по любой колонке: ID, модель, производитель, цвет, спальное место, высота, длина, материал, основание, конструкция, изголовье, количество, цена.
- **Импорт и экспорт CSV** — совместимость с Microsoft Excel, корректная работа с русскими буквами.
- **Мягкое удаление** — товары скрываются вместо полного удаления, восстановление в один клик.
- **Автосохранение** — напоминание о сохранении изменений при выходе из приложения.
- **Единый дизайн** — десктоп и мобильное приложение выглядят и работают одинаково.

---

## Быстрый старт

### Windows
1. Скачайте ZIP-архив или RAR-архив.
2. Распакуйте в любую папку.
3. Запустите `МебельТорг.exe`. Java не требуется.

### Android
1. Скачайте APK-файл.
2. Разрешите установку из неизвестных источников.
3. Установите и откройте приложение.

---

## Как работать

1. **Импортируйте** CSV-файл с данными или начните с пустого каталога.
2. **Добавляйте, редактируйте, удаляйте(скрывайте)** товары.
3. **Фильтруйте и сортируйте** каталог под текущую задачу.
4. **Сохраняйте** изменения в CSV перед выходом.
5. Переносите CSV-файл между устройствами — десктоп и телефон используют один формат.

---

## Технологии

- **Язык:** Kotlin
- **Десктоп:** Swing (JVM), сборка через jpackage в автономный EXE с JRE.
- **Android:** Jetpack Compose, Material Design 3.
- **Данные:** CSV с BOM для совместимости с Excel.
