МебельТорг

Учёт кроватей — просто и удобно

📦 Все версии (Windows и Android)

О приложении

«МебельТорг» — это бесплатное приложение для учёта кроватей. Управляйте каталогом, ведите складской учёт, фильтруйте и сортируйте товары, импортируйте и экспортируйте данные в Excel. Работает на Windows и Android с единым файлом данных.

Возможности

📋

Полный каталог

14 полей: модель, производитель, цвет, размеры, материал, конструкция и другие.

🔍

Фильтры и поиск

Фильтрация по 10 параметрам, поиск по модели, сортировка по любой колонке.

📁

Импорт / Экспорт

Загрузка и выгрузка CSV-файлов, совместимых с Microsoft Excel.

🔄

Единые данные

Один CSV-файл для десктопа и телефона. Полная совместимость.

👻

Мягкое удаление

Скрывайте товары вместо полного удаления. Восстановление в один клик.

💾

Автосохранение

Напоминание о сохранении изменений при выходе из приложения.
